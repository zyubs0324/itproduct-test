# Website Monitoring Dashboard

## Overview

A single-page dashboard for a website monitoring service. It provides users with a real-time overview of their monitored websites, including uptime, performance, and any detected issues.

## Design & Features

### Style & Design
- **Aesthetics:** Modern, clean, and intuitive interface with a focus on data visualization.
- **Layout:** Responsive design that adapts to different screen sizes (mobile and web).
- **Color Palette:** A vibrant and energetic color palette with a wide range of hues to differentiate between statuses (e.g., green for 'up', red for 'down', yellow for 'issues').
- **Typography:** Expressive and clear typography to create a visual hierarchy. Hero text for key metrics, section headlines, and legible data in tables and charts.
- **Iconography:** Use of icons to enhance understanding and navigation.
- **Effects:** Subtle use of multi-layered drop shadows to create depth and a "lifted" feel for cards and interactive elements. A soft "glow" effect on interactive elements to indicate their state.
- **Texture:** A subtle noise texture on the main background for a premium, tactile feel.

### Features
- **Navigation:** An intuitive navigation bar for easy access to different sections of the dashboard.
- **Summary Cards:** Cards displaying key metrics at a glance (e.g., total websites, uptime percentage, number of issues).
- **Real-time Chart:** A chart visualizing real-time data, such as response times or uptime over a period.
- **Monitored Websites Table:** A detailed table listing all monitored websites with their status, response time, and last check time.
- **Accessibility:** The dashboard will be designed with accessibility in mind, following a11y standards to be usable by a wide range of users.
